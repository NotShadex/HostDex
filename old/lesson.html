Introduction

A concise, practical reference for learning PowerShell. Each section contains a short explanation followed by example commands you can run in the PowerShell console. Run PowerShell as Administrator when a command requires elevated privileges.

PowerShell Basics

Cmdlets are the basic commands in PowerShell. They follow a Verb-Noun pattern (for example, Get-Process). PowerShell works with objects rather than plain text, which enables piping and structured queries.

Common informational cmdlets:

Get-Verb
Get-Command
Get-Member
Get-Help

Check PowerShell version and get help for commands:

$PSVersionTable
Get-Help Get-ChildItem

List files including ONLY hidden:

Get-ChildItem -Hidden

List files including hidden:

Get-ChildItem -Force

System Inspection

Use these commands to discover available commands and inspect specific cmdlet information.
Get-Command
Get-Command Get-ChildItem

Managing Services

Services have states such as Running, Stopped, and Paused. Use Get-Service to list and query services, and pipe output to filter or select properties.
Get-Service
Get-Service -Name *time*
You have multiple choices for selecting services:
Get-Service | Where-Object {$_.Status -eq "Running"} OR 
Get-Service | Where-Object -Property Status -eq "Running"
Example for selecting multiple items
Get-Service | Select-Object Name, StartType
Get-Service -Name *w32time* | Select-Object Name, Status, StartType

Start / Stop / Restart Services

Before changing services, use -WhatIf to simulate the action. Use -Confirm to prompt for confirmation.
Start-Service -Name w32time -WhatIf
Start-Service -Name w32time -Confirm
Get-Service -Name w32time | Select-Object Name, Status
Get-Service -Name w32time
Stop-Service -Name w32time -Confirm
Restart-Service -Name w32time -Confirm

Service Startup Types

Startup types determine whether a service starts automatically, manually, or is disabled.
Set-Service -Name w32time -StartupType Disabled
Set-Service -Name w32time -StartupType Automatic -Confirm
Set-Service -Name w32time -Status Stopped -StartupType Manual -PassThru
Get-Service -Name w32time | Select-Object Name, DisplayName, Status, StartType

Process Management

List processes, filter by resource usage, and stop or retrieve details for specific processes.
Get-Process
Get-Process -Name notepad++
Get-Process -Name *note*
Get-Process -Name notepad++ | Select-Object Name, ID, WS, StartTime
Get-Process | Where-Object {$_.WS -gt 250MB}
Stop-Process -Name notepad++
Stop-Process -Id 2180 -Confirm
Get-Process notepad++ | Stop-Process

Starting Applications

Start processes either normally or elevated. -Verb RunAs requests elevation.
Start-Process "C:\Program Files\Notepad++\notepad++.exe"
Start-Process "C:\Program Files\Notepad++\notepad++.exe" -Verb RunAs
Start-Process "C:\Program Files\Notepad++\notepad++.exe" -Verb RunAsUser

Network Adapters & IP

Inspect adapters, physical devices, IP addresses and control adapter state.
Get-NetAdapter
Get-NetAdapter -Physical
Get-NetIPAddress -AddressFamily IPv4
Get-NetIPAddress -AddressFamily IPv6
ipconfig /all
Get-NetAdapter | Select Name, MacAddress, LinkSpeed
Restart-NetAdapter -Name Ethernet -Confirm -PassThru
Disable-NetAdapter -Name Ethernet -Confirm -PassThru
Enable-NetAdapter -Name Ethernet -Confirm -PassThru

Event Logs

Query Windows event logs, filter by type and retrieve recent entries. Useful for troubleshooting.
Get-EventLog -LogName System | more
Get-EventLog -LogName System -Newest 20
Get-EventLog -LogName System -Newest 2 | Select-Object Index, TimeWritten, EntryType, InstanceId, Message
Get-EventLog -LogName System -EntryType Error,Information,FailureAudit,SuccessAudit,Warning
Get-Content C:\Users\YourUser\Desktop\yourlog.log -TotalCount 5
Get-Content C:\Users\YourUser\Desktop\yourlog.log -Tail 5
Get-Content yourlog.log | more

Start with the essentials

Understanding Cmdlets

Cmdlets are PowerShell’s built-in commands. They always follow a Verb-Noun format such as Get-Process.

Working with Objects

PowerShell outputs objects, not plain text. This enables filtering, sorting, and piping data with precision.

Navigating the Shell

Use PowerShell like a file system with commands like Get-ChildItem, Set-Location, and New-Item.

Quick Sneak Peak

Cmdlets

This is an example of the format our tutorials will follow.

    # List files
    Get-ChildItem

    # Show running processes
    Get-Process

    # Get system services
    Get-Service

    # Find commands by name
    Get-Command *network*

    # Filter results
    Get-Process | Where-Object {$_.CPU -gt 10}

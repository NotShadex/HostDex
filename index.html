📟 Commands & built-ins

Bash commands are powerful executables or built-ins. Combine ls, grep, awk with pipes (|) for true magic.

$> ls -la | grep ".sh"
$> echo "hello world" > output.txt

🔗 Pipes & Redirection

Unleash the power of pipelines. Redirect stdout, stderr, and chain programs for complex data processing.

$> ps aux | grep bash | wc -l
$> find . -name "*.log" 2>/dev/null

📁 Navigating the filesystem

Master cd, pwd, mkdir, rm -rf (carefully!) and glob patterns.

$> cd /var/log && tail -f syslog
$> for d in */; do echo $d; done

📜 Essential Bash Snippets

Real commands to kickstart your shell scripting journey. Each snippet follows bash best practices.

$> # List all files with details
$> ls -lah --color=auto

$> # Show running processes (bash-centric)
$> ps -ef | grep -E "bash|zsh"

$> # Monitor disk usage
$> df -h | grep '^/dev'

$> # Find commands by keyword in PATH
$> compgen -c | grep 'net'

$> # Filter processes using awk
$> ps aux | awk '$3 > 2.0 {print $2, $11}'

$> # one-liner loop: backup configs
$> for f in *.conf; do cp "$f" "${f}.bak"; done

🧪 Variables & conditionals

Bash scripting fundamentals: variables, test brackets, and exit codes.

$> name="BashNinja"
$> echo "Hello, $name"

$> if [ -f "/etc/passwd" ]; then
  >    echo "passwd exists"
  > fi

$> # command substitution
$> today=$(date +%Y-%m-%d)
$> echo "backup-$today.tar.gz"

⚙️ Functions & automation

Create reusable functions and schedule scripts with cron.

$> greet() { echo "⚡ Bash is $1"; }
$> greet "powerful"

$> # cron-style backup alias
$> alias backup_home='tar -czf backup.tar.gz ~/Documents'

$> # exit on error
$> set -e; ./deploy.sh || echo "failed"

📘 Shebang & portability

Start every script with #!/usr/bin/env bash for robust execution across environments. Combine with set -euo pipefail for bulletproof scripts.

$ bash --version | head -1

📦 Package management (apt/brew)

Learn to manage tools natively — apt install or brew install for Linux/macOS.

$> sudo apt update && sudo apt install neofetch
$> neofetch

📝 Environment & dotfiles

Customize your .bashrc, .profile, or .bash_aliases. Boost your terminal game.

$> alias ll='ls -alF'
$> export PATH="$HOME/bin:$PATH"

🧹 Text processing power

sed, awk, cut, sort — the classic Unix text tools. Extract logs, transform CSV.

$> cat access.log | awk '{print $1}' | sort | uniq -c
